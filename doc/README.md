## 当前能力边界

- 已实现：邮箱密码认证、可选 GitHub OAuth、数据库会话、浏览器本地聊天历史、DeepSeek 流式对话、百度千帆联网搜索、百度图片文字识别、只读管理后台。
- 部分实现：数据库已包含云端会话、模型、Prompt、工具、用量和审计模型，后台可读取其中部分数据，但前台聊天尚未写入这些表。
- 尚未实现：聊天云同步、后台增删改、知识库/RAG、通用 HTTP Tool 执行器、多模型动态目录、计费写入与审计写入。
